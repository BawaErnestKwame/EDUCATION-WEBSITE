HTML INTRODUCTION

HTML stands for Hypertext Markup Language, and it is the most widely used language to write Web Pages.
Hypertext refers to the way in which Web pages (HTML documents) are linked together. Thus, the link available on a webpage is called Hypertext.

As its name suggests, HTML is a Markup Language which means you use HTML to simply "mark-up" a text document with tags that tell a Web browser how to structure it to display. Originally, HTML was developed with the intent of defining the structure of documents like headings, paragraphs, lists, and so forth to facilitate the sharing of scientific information between researchers. Now, HTML is being widely used to format web pages with the help of different tags available in HTML language.

In its simplest form, following is an example of an HTML document:

<!DOCTYPE html>
<html>
<head>
<b></b><title>This is document title</title>
</head>
<body>
<h1>This is a heading</h1>
<p>Document content goes here.....</p>
</body>
</html>

Here is the Output

This is a Heading

Document content goes here.....

HTML tags

HTML tags are the building blocks of HTML (HyperText Markup Language), used to structure content on a webpage. Each HTML tag defines a different element or part of the document, such as text, images, links, and more.

Tages Description
<!DOCTYPE...> This tag defines the document type and HTML version.
<html > This tag encloses the complete HTML document and mainly comprises of document header which is represented by <head>...</head> and document body which is represented by <body>...</body> tags</td>
<head> This tag represents the document's header which can keep other HTML tags like <title>, <link> etc.</td>
<body> This tag represents the document's body which keeps other HTML tags like <h1>, <div>, <p> etc. </td>
<h1> This tag represents the heading.
<p>

This tag represents a paragraph.

HTML Links

HTML links, also called hyperlinks, allow users to navigate from one webpage to another or to different sections within the same page. Links are created using the <a> (anchor) tag./p>

HTML Links - Hyperlinks

HTML links are hyperlinks.You can click on a link and jump to another document.

When you move the mouse over a link, the mouse arrow will turn into a little hand.

Note: A link does not have to be text. A link can be an image or any other HTML element!

<a>: The anchor tag that defines the link.

href: The href attribute specifies the destination URL (the page or resource the link points to).

Clickable text: The text inside the anchor tag that users can click to follow the link.

DEFFERENCE TYPES OF LINKS

External Links: It direct users to other websites.

<a> href="https://www.google.com">Go to Google </a>

Internal Links: Direct users to another page on the same website.

<a> href="about.html">About Us </a>

Anchor Links: Jump to a specific section within the same page.

<a> href="#section1">Go to Section 1 </a>

Email Links: Open the user's email client to send an email.

<a> href="#section1">Go to Section 1 </a>

Note: Links are an essential part of website navigation and help connect different pages and resources on the web
